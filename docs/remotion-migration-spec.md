# Remotion Migration Spec

**Version:** 2.0  
**Replaces:** Legacy per-scene Python renderer + all `templates/scenes/*.py`  
**Everything else stays the same** — Claude service, ElevenLabs, FFmpeg, job queue, brand configs, API endpoints.

---

## Visual Philosophy — "Mixed-Media Documentary"

Every video should feel like it was touched by a human, not generated by a machine. The goal is the Vox/documentary aesthetic — organic textures, physical imperfection, kinetic energy. Three principles drive every design decision:

1. **Depth over flatness** — every scene has at least three layers: background texture, primary content, overlay (grain/vignette). Never a flat color with text on top.
2. **Organic motion** — spring physics only. No linear easing anywhere. Elements should feel like they have weight and snap.
3. **Designed imperfection** — subtle grain, slight element jitter, paper textures. The video should feel crafted, not rendered.

### V1 (Build Now)

- ✅ Film grain overlay — global, persists across every scene
- ✅ Vignette overlay — global, darkens edges of every scene
- ✅ Paper/noise texture backgrounds — replaces flat solid colors
- ✅ Spring physics on all animations — `Easing.out(Easing.back())` everywhere
- ✅ `VoxDocumentary` scene — newspaper clipping with SVG animated highlighter
- ✅ `KineticChart` scene — self-drawing bar charts via Recharts

### V2 (Future — do not build yet)

- 🔜 **Lottie animations** — curate 15-20 topic animations manually from LottieFiles.com, map to topic categories in brand config. Do NOT use the LottieFiles API (inconsistent quality, unreliable keyword matching).
- 🔜 **CollageHero strobe** — rapid image montage hook. Requires a curated per-brand image library. Unsplash API keyword search produces generic stock that degrades quality. Only build once you have time to curate proper image sets per brand.
- 🔜 **Audio layering** — background lofi track with auto-ducking + paper rustle SFX on document scenes.

## 4. Global Visual Layer

These components wrap every scene automatically via `SceneWrapper`. Cursor must apply `SceneWrapper` in `VideoComposition.tsx` — individual scene components never import these directly.

### `src/global/FilmGrain.tsx`

A persistent animated noise texture that overlays the entire frame. Makes everything feel cinematic and organic.

```tsx
import { useCurrentFrame, AbsoluteFill } from 'remotion';

interface Props {
	opacity?: number; // 0.03–0.05 recommended
}

export const FilmGrain: React.FC<Props> = ({ opacity = 0.04 }) => {
	const frame = useCurrentFrame();
	// Shift the noise texture every frame to animate the grain
	const offsetX = (frame * 97) % 200;
	const offsetY = (frame * 53) % 200;

	return (
		<AbsoluteFill style={{ pointerEvents: 'none', zIndex: 100 }}>
			<div
				style={{
					position: 'absolute',
					inset: 0,
					opacity,
					backgroundImage: `url('/textures/noise-dark.png')`,
					backgroundSize: '200px 200px',
					backgroundPosition: `${offsetX}px ${offsetY}px`,
					mixBlendMode: 'screen'
				}}
			/>
		</AbsoluteFill>
	);
};
```

**Noise texture:** Generate `public/textures/noise-dark.png` and `noise-light.png` using any noise generator. A 200×200px tileable Perlin noise PNG at 60% contrast works well. Alternatively use an SVG `feTurbulence` filter rendered to canvas and saved as PNG.

### `src/global/SceneWrapper.tsx`

Wraps every scene with grain + vignette + the textured background. This is applied in `VideoComposition.tsx` — scenes themselves render on top of this.

```tsx
import { AbsoluteFill } from 'remotion';
import { FilmGrain } from './FilmGrain';
import { Vignette } from './Vignette';
import { BrandTheme } from '../types';

interface Props {
	brand: BrandTheme;
	children: React.ReactNode;
}

export const SceneWrapper: React.FC<Props> = ({ brand, children }) => {
	return (
		<AbsoluteFill>
			{/* Layer 1: Textured background — never a flat color */}
			<AbsoluteFill
				style={{
					backgroundColor: brand.colors.background,
					backgroundImage: `url('/textures/noise-dark.png')`,
					backgroundSize: '300px 300px',
					backgroundRepeat: 'repeat',
					opacity: 0.08 // very subtle — just enough to break the flatness
				}}
			/>

			{/* Layer 2: The scene content */}
			<AbsoluteFill>{children}</AbsoluteFill>

			{/* Layer 3: Vignette + grain on top of everything */}
			<Vignette />
			<FilmGrain opacity={0.04} />
		</AbsoluteFill>
	);
};
```

### `src/components/AnimatedText.tsx`

```tsx
import { useCurrentFrame, useVideoConfig, interpolate, Easing } from 'remotion';

interface Props {
	children: React.ReactNode;
	style?: React.CSSProperties;
	animationStyle?: string; // "fade_up" | "slide_in" | "fade_in" | "typewrite"
	delay?: number; // frames to wait before starting
}

export const AnimatedText: React.FC<Props> = ({
	children,
	style,
	animationStyle = 'fade_up',
	delay = 0
}) => {
	const frame = useCurrentFrame();
	const f = Math.max(0, frame - delay);

	let opacity = 1;
	let translateY = 0;
	let translateX = 0;

	if (animationStyle === 'fade_up') {
		opacity = interpolate(f, [0, 20], [0, 1], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.ease)
		});
		translateY = interpolate(f, [0, 20], [30, 0], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.ease)
		});
	} else if (animationStyle === 'fade_in') {
		opacity = interpolate(f, [0, 20], [0, 1], { extrapolateRight: 'clamp' });
	} else if (animationStyle === 'slide_in') {
		opacity = interpolate(f, [0, 15], [0, 1], { extrapolateRight: 'clamp' });
		translateX = interpolate(f, [0, 20], [40, 0], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.back(1.2))
		});
	} else if (animationStyle === 'slide_up') {
		opacity = interpolate(f, [0, 20], [0, 1], { extrapolateRight: 'clamp' });
		translateY = interpolate(f, [0, 20], [50, 0], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.back(1.5))
		});
	}

	return (
		<div
			style={{
				opacity,
				transform: `translateY(${translateY}px) translateX(${translateX}px)`,
				...style
			}}
		>
			{children}
		</div>
	);
};
```

interface Props {

### `src/components/HighlightRect.tsx`

```tsx
import { useCurrentFrame, interpolate, Easing } from 'remotion';

interface Props {
	color: string;
	opacity?: number;
	delay?: number;
	animationStyle?: 'slide_in' | 'stamp' | 'fade_in';
	style?: React.CSSProperties;
	children?: React.ReactNode;
}

export const HighlightRect: React.FC<Props> = ({
	color,
	opacity = 0.15,
	delay = 0,
	animationStyle = 'fade_in',
	style,
	children
}) => {
	const frame = useCurrentFrame();
	const f = Math.max(0, frame - delay);

	let rectOpacity = opacity;
	let scale = 1;
	let translateX = 0;

	if (animationStyle === 'stamp') {
		scale = interpolate(f, [0, 8], [1.08, 1], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.back(2))
		});
		rectOpacity = interpolate(f, [0, 6], [0, opacity], { extrapolateRight: 'clamp' });
	} else if (animationStyle === 'slide_in') {
		translateX = interpolate(f, [0, 20], [-60, 0], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.ease)
		});
		rectOpacity = interpolate(f, [0, 10], [0, opacity], { extrapolateRight: 'clamp' });
	} else {
		rectOpacity = interpolate(f, [0, 15], [0, opacity], { extrapolateRight: 'clamp' });
	}

	return (
		<div
			style={{
				backgroundColor: color,
				opacity: rectOpacity,
				transform: `scale(${scale}) translateX(${translateX}px)`,
				position: 'relative',
				...style
			}}
		>
			{children}
		</div>
	);

interface Props {
	color?: string;
	delay?: number;
	width?: string | number;
	thickness?: number;

import { AccentBar } from '../components/AccentBar';

export const KineticChart: React.FC<SceneProps> = ({ scene, brand }) => {
	const frame = useCurrentFrame();
	const { heading, data, unit = '', context } = scene.content as any;
	const accentHex = resolveColor(scene.accent_color, brand);

	// Each bar animates in with a staggered delay
	// Recharts doesn't natively animate in Remotion — we fake it by
	// scaling the data values based on the current frame
	const BAR_DELAY = 8; // frames between each bar starting
	const BAR_DURATION = 20; // frames for each bar to reach full height
	const CHART_START = 20; // heading animates first

	const animatedData = data.map((item: any, i: number) => {
		const barStart = CHART_START + i * BAR_DELAY;
		const f = Math.max(0, frame - barStart);
		const progress = interpolate(f, [0, BAR_DURATION], [0, 1], {
			extrapolateRight: 'clamp',
			easing: Easing.out(Easing.back(1.2))
		});
		return { ...item, animatedValue: item.value * progress };
	});

	return (
		<AbsoluteFill
			style={{
				backgroundColor: brand.colors.background,
				flexDirection: 'column',
				padding: '80px 60px',
				justifyContent: 'center'
			}}
		>
			{/* Heading */}
			<AnimatedText
				animationStyle="fade_up"
				style={{
					fontFamily: brand.fonts.heading,
					fontSize: 48,
					fontWeight: 800,
					color: accentHex,
					marginBottom: 12
				}}
			>
				{heading}
			</AnimatedText>
			<AccentBar color={accentHex} width={140} height={3} delay={12} style={{ marginBottom: 48 }} />

			{/* Chart */}
			<div style={{ width: '100%', height: 500 }}>
				<ResponsiveContainer width="100%" height="100%">
					<BarChart data={animatedData} margin={{ top: 20, right: 20, left: 0, bottom: 20 }}>
						<XAxis
							dataKey="label"
							tick={{ fill: brand.colors.muted, fontFamily: brand.fonts.body, fontSize: 22 }}
							axisLine={{ stroke: brand.colors.muted, strokeWidth: 1 }}
							tickLine={false}
						/>
						<YAxis hide />
						<Bar dataKey="animatedValue" radius={[6, 6, 0, 0]} maxBarSize={120}>
							{animatedData.map((entry: any, index: number) => (
								<Cell key={index} fill={resolveColor(entry.color ?? 'accent', brand)} />
							))}
						</Bar>
					</BarChart>
				</ResponsiveContainer>
			</div>

			{/* Value labels — appear after bars are mostly done */}
			{context && (
				<AnimatedText
					animationStyle="fade_in"
					delay={CHART_START + data.length * BAR_DELAY + 10}
					style={{
						fontFamily: brand.fonts.body,
						fontSize: 26,
						color: brand.colors.muted,
						textAlign: 'center',
						marginTop: 24
					}}
				>
					{context}
				</AnimatedText>
			)}
		</AbsoluteFill>
	);
};
```

### Remaining Scenes — Implementation Notes

Build these following the same pattern. Each gets its own file in `src/scenes/`.

**`TextReveal.tsx`** — Side rule draws top to bottom, body text fades in beside it. Font size scales with text length (long text = smaller font).

**`QuoteCard.tsx`** — Giant `❝` behind quote at 8% opacity. Short accent bar above attribution. `dramatic_pause` = delay entire reveal by 15 frames.

**`ComparisonTable.tsx`** — Two column layout. Left header red background, right header accent background. Rows fade in via LaggedStart pattern (stagger each row by 8 frames).

**`ClosingCard.tsx`** — Full-width accent bar at top draws left to right. Heading fades up. CTA in a bordered rounded rect (border: `2px solid accent`, padding, rounded corners). URL in muted below.

**`Checklist.tsx`** — Items cascade. Each item: empty square border appears → text slides in → square fills with checkmark. 12 frames per item.

**`MechanismDiagram.tsx`** — Nodes as rounded rects. Arrows between them using `interpolate` to animate width from 0 to full. Node 1 appears, arrow draws, node 2 appears, etc.

**`EvidenceStack.tsx`** — Cards slide in from right and stack. Use `zIndex` and slight vertical offset per card. Earlier cards scale to 0.92 as new cards arrive.

**`ObjectionRebuttal.tsx`** — Same two-act pattern as `MythVsReality` but gray border for objection, accent border for response. Objection text uses typewriter effect (reveal characters using string slice based on frame count).

## 10. Python Renderer (`services/remotion_renderer.py`)

Replaces the old `template_renderer.py` / per-scene subprocess pipeline. Calls Remotion CLI via subprocess.

```python
"""Remotion renderer — replaces template_renderer.py"""

from __future__ import annotations

import json
import os
import subprocess
from pathlib import Path
from typing import Any

REMOTION_DIR = Path(__file__).resolve().parent.parent / "video-templates"


class RemotionRenderer:
    def __init__(self, brand_config: dict[str, Any], quality: str, output_dir: str):
        self.brand = brand_config
        self.quality = quality
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(parents=True, exist_ok=True)

    def _quality_to_crf(self) -> int:
        return {"low": 28, "medium": 18, "high": 12}.get(self.quality, 18)

    def _build_brand_prop(self) -> dict:
        """Convert brand config JSON → VideoProps.brand shape."""
        c = self.brand.get("colors", {})
        return {
            "brandId":     self.brand.get("brand_id", "default"),
            "displayName": self.brand.get("display_name", ""),
            "colors": {
                "background":  c.get("background", "#0a0a0f"),
                "primaryText": c.get("primary_text", "#f0f0f5"),
                "accent":      c.get("accent", "#2563eb"),
                "gold":        c.get("gold", "#f59e0b"),
                "muted":       c.get("muted", "#6b7280"),
            },
            "fonts": {
                "heading": self.brand.get("fonts", {}).get("heading", "Montserrat"),
                "body":    self.brand.get("fonts", {}).get("body", "Lato"),
            },
        }

    def render_video(self, scenes: list[dict], job_id: str) -> str:
        """
        Render all scenes as a single video via Remotion CLI.
        Returns path to output MP4.
        """
        output_path = self.output_dir / job_id / "video_no_audio.mp4"
        output_path.parent.mkdir(parents=True, exist_ok=True)

        props = {
            "scenes": scenes,
            "brand":  self._build_brand_prop(),
            "fps":    30,
        }

        result = subprocess.run(
            [
                "npx", "remotion", "render",
                "VideoComposition",
                str(output_path),
                "--props", json.dumps(props),
                "--crf", str(self._quality_to_crf()),
                "--codec", "h264",
            ],
            cwd=str(REMOTION_DIR),
            capture_output=True,
            text=True,
            timeout=600,
        )

        if result.returncode != 0:
            raise RuntimeError(
                f"Remotion render failed:\n{result.stderr[-2000:]}"
            )

        return str(output_path)
```

## 13. Key Rules for Cursor

1. **Every animation uses `useCurrentFrame()` + `interpolate()`** — never CSS transitions, never `setTimeout`, never `useEffect` for animation. Remotion is frame-based.
2. **`interpolate()` always needs `extrapolateRight: "clamp"`** — without it values go past their target.
3. **`delay` params are in frames** — at 30fps, 30 frames = 1 second.
4. **`Series.Sequence` handles scene timing** — don't calculate frame offsets manually. Each scene component sees frames starting from 0.
5. **Fonts must be loaded via `@remotion/google-fonts`** — Remotion renders in a headless browser, so system fonts aren't available. Use `@remotion/google-fonts` for Google Fonts.
6. **Width is 1080, height is 1920** (portrait 9:16 for YouTube Shorts / social). Change Root.tsx to 1920×1080 for landscape.
7. **Build and test in Remotion Studio first** — `npx remotion studio` gives a live preview. Don't test via Python service until each component looks right in Studio.
8. **`resolveColor()` always** — never hardcode hex in scene components.
9. **The Python renderer passes all scenes at once** — Remotion sequences them internally via `Series`. No per-scene subprocess calls.
10. **`Easing.out(Easing.back(n))`** is your friend for Apple-style snap — use it on anything that should feel physical and satisfying.
11. **`SceneWrapper` is mandatory** — every scene in `VideoComposition.tsx` must be wrapped in `<SceneWrapper brand={brand}>`. Never skip it. This is what applies grain, vignette, and texture globally.
12. **Never use flat background colors in scenes** — backgrounds come from `SceneWrapper`'s textured layer. Individual scene components set their own overlapping elements but don't need to fill the background.
13. **The jitter effect** — for `VoxDocumentary` and any "physical document" style scenes, add a subtle `Math.sin(frame * rate) * amplitude` on translateX and rotate to simulate hand-placed elements. Keep amplitude under 2px and rotation under 0.5deg.
14. **Recharts in Remotion** — Recharts doesn't use Remotion's frame system natively. Always fake animation by mapping `frame` to scaled data values via `interpolate()`, not by using Recharts' built-in animation props (which don't work in headless render).
15. **Noise textures must be real PNGs** — generate `public/textures/noise-dark.png` and `noise-light.png` before building any scene. A 200×200px tileable Perlin noise PNG. Without these files, `SceneWrapper` will silently show a flat background.
16. **`vox_documentary` and `kinetic_chart` must be added to `claude_service.py` ANIMATION_BY_TYPE** — then run `python generate_scene_animations_ts.py` to sync the TS file.
